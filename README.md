A modern food discovery and delivery platform built with React, TypeScript, and Tailwind CSS.

## Features

- **Restaurant Discovery** - Browse 6 restaurants across categories (Pizza, Sushi, Chinese, Healthy, Burgers, Cafe)
- **Real-time Search** - Filter restaurants by name or cuisine tags
- **Category Filtering** - Horizontal scrollable category selector
- **Shopping Cart** - Slide-over sidebar with quantity controls and automatic totals
- **Mock Authentication** - Sign in modal with simulated login flow
- **Multi-step Checkout** - Address entry ‚Üí Payment (mock credit card UI) ‚Üí Order confirmation
- **Order Tracking** - Animated progress stepper showing order status
- **Mobile Optimized** - Responsive design with sticky "View Cart" button for mobile

## Tech Stack

- React 18
- TypeScript
- Tailwind CSS
- Lucide React (icons)
- Vite

## Getting Started

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```
